Robot Generator

July 2015 - Ruby, Rails, Heroku

Robot Generator is a Ruby on Rails application developed for the Code Fellows Development Accelerator application. I learned how to generate a Rails app from the Treehouse tutorial and adapted their tutorial for my app. Special features: each new robot is assigned a unique image from robohash.org, based on the "name" field, which is the only required field. After deploying, I updated the validation settings to ensure that if robot arms value is given, it cannot be a negative number. I also added some simple css for improved usability.

Achievement Bot

April 2015 - Javascript, Node.js, Heroku, Twitter

@unlockachievemt is a Twitter bot that celebrates achievements. It retweets the latest tweet using the "#achievementunlocked" hashtag (with some filters to avoid game-generated auto-tweets). It attempts to retweet once per hour. I deployed the bot using Heroku, Node.js, and json adapted from various web sources including Darius K's 'example bot' and Sarah K's 'twitter bot with node js' write-up.

Deck of Cards, Play War!

February 2015 - HTML, CSS, Javascript, JQuery

Deck of cards was my final assignment for Code Fellows Foundations I. The page uses Javascript to iterate a deck of cards and JQuery to shuffle them. JQuery assigns classes to the card objects based on their suit and CSS assigns an appropriate image. I added play functionality for a game of "War" against the computer. The game has a flaw wherein it never seems to end (this is not good game design, obviously, but parent friends tell me this is a 'feature').

Number Guessing Game

February 2015 - HTML, JavaScript

Player guesses a secret number between 1 and 100, the game keeps track of your high and low to narrow the range and reports back when you've won with how many guesses it took. Initially used 'prompt' and 'alert' to interact with player. Simple little JavaScript game, first assignment of Code Fellows Foundations I class (later built out with some html, JavaScript and css).

Bletchley Bot

January 2015 - Python, Heroku, Twitter

More than 75% of Bletchley Park's code-cracking veterans are women. Using Python, I created a bot that tweets their names and service summaries every 30 minutes. Through this process I practiced using Python to manipulate .csv data and communicate with twitter's API. I also learned to prep and deploy apps using git and Heroku. This was one of my first coding projects, and still a favorite.
